Plugins

  • Excalidraw
  • Hexo Toolkit
  • Shell commands

为了让你找回 Typora 的感觉，请安装以下插件：

插件名称 作用 推荐配置
Editing Toolbar 核心推荐。在顶部增加类似 Word/Typora 的格式工具栏。 必装，完美复刻 Typora 菜单。
Image Toolkit 点击图片全屏查看、缩放、滚轮放大。 解决 Obsidian 默认无法点击看大图的问题。
Easy Typing 优化中文输入体验，自动在中英文之间加空格。 强迫症福音。
Dataview 在首页生成一个“最近更新的文章”列表。 把 Obsidian 首页变成你的博客 Dashboard。

Theme

Obsidian 的默认主题是给程序员看的，太素了。

  • 操作： 设置 -> 外观 (Appearance) -> 主题 (Themes) -> 管理 (Manage)。

  • 推荐主题（专治简陋）：

    • Blue Topaz： （强烈推荐）这是国人开发的主题，功能极其强大，色彩丰富，默认就很漂亮，适合习惯 Typora 的用户。

    • Minimal： 极简主义，Apple 风格，干净利落。

    • Primary： 暖色调，字体排版非常舒服，像杂志一样。

  • Theme: Minimal
  • Plugin: Minimal Theme Settings
    • Flexio
