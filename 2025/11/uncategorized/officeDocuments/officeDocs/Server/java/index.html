相关命令

  • assemble：编译打包该工程
  • build： 编译打包并测试该工程
  • buildDependents: 编译打包并测试该工程和依赖它的工程
  • buildNeeded：编译打包并测试该工程和它依赖的工程
  • classes：编译打包主要的类文件
  • clean：删除编译目录
  • jar：编译成jar包，该jar包包含主要的类文件
  • testClasses：编译打包该测试文件

Annotation

  1. The annotation member types must be one of:

    • primitive
    • String
    • an Enum
    • another Annotation
    • Class
    • an array of any of the above

    It does seem restrictive, but no doubt there are reasons for it.

    Also note that multidimensional arrays (e.g. String[][]) are implicitly forbidden by the above rule.
