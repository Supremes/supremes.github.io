SourceCode

The copyright notice must be included in each file. Source code files must additionally include the designation “Confidential & Proprietary”, as shown on the right.

If you have received permission from the Legal Department to make certain code files available under an open source license (e.g., MIT, BSD-style), the “All Rights Reserved. Confidential & Proprietary” label should be omitted from the copyright notice.

Opensource:

Copyright © 20xx. Cloud Software Group, Inc. All Rights Reserved.

Non-opensource:

Copyright © 20xx. Cloud Software Group, Inc. All Rights Reserved. Confidential & Proprietary.

Executal Software Code

The copyright notice must appear on an opening executable code display (e.g., a splash screen that appears each time the software is used) as well as in any “readme.txt” file that is delivered along with the software. For software distributed on physical media (e.g., a CD-ROM), the copyright notice additionally must be printed on an external label.

  • splash screen
  • readme.txt (It’s deliverd along with the software)
    Copyright © 20xx. Cloud Software Group, Inc. All Rights Reserved.

Product Documentation

All product documentation (e.g., readme.txt files, release notes, web help, and product guides) must bear the Cloud Software Group copyright notice. Product documentation that is not made publicly available must include the designation “Confidential Information”, as shown on the right.

  • readme
  • release notes
  • Web help
  • product guides

The public version:

Copyright © 20xx. Cloud Software Group, Inc. All Rights Reserved.

Only for internal:

Copyright © 20xx. Cloud Software Group, Inc. All Rights Reserved. Confidential & Proprietary.

Video

All Cloud Software Group-owned videos must display the Cloud Software Group copyright notice. The notice shown on the right should appear at or close to the beginning of the video or at the end. You should also include the copyright notice on the website that hosts the video.

If the video is for internal Cloud Software Group use only or is available to a viewer only pursuant to the terms of an agreement with the viewer, “Confidential & Proprietary” must be added to the end of the notice.
